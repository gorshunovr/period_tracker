## v1.0

- Initial public release
- Multi-user profile support with per-person cycle settings
- Event logging (period starts and symptoms) with date picker
- Cycle predictions (next period, PMS, ovulation, fertile window)
- Statistical learning from logged cycle history
- Daily digest across all profiles
- Cycle statistics view with regularity and confidence
- Optional 4-digit PIN protection with lockout
- Local-only storage on SD card (CSV + profile files)
